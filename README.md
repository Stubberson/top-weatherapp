# Weather Application
An intermediate JS project for the Odin Project on how to use APIs and asynchronous code.

Weather data is retrieved from [Visual Crossing API](https://www.visualcrossing.com/weather-api/). Weather icons used are from [Google's Weather API](https://developers.google.com/maps/documentation/weather/weather-condition-icons).

This project, for me, was a deep dive into SVGs and how to create nice visuals using data fetched from an API. I decided to stay away from external libraries (like [Chart.js](http://chartjs.org/)) and work only with native elements.

My aim was to minimize the user's need to read different weather statistics as text, turning data into intuitive and informative visual signals.
